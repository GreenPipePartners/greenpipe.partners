Training flow

From workstation setup to a site-specific project.

  1. Day 1

    Set up the engineering workstations

    Configure computers, OpenCode, and AI-assisted host applications while reviewing the software stack and where each tool fits.

  2. Day 2

    Build each attendee's Ubuntu VM lab

    Stand up a personal Ubuntu VM with containerized Ignition and SQL Server environments while reviewing Docker and WSLC workflows.

  3. Day 3

    Apply the lab to your site

    Implement a practical project tailored to your facility, controls standards, and near-term engineering needs.

  4. Day 4 (Optional)

    Build a Rockwell Windows VM with AI agents

    Use AI agents to help build out a Windows VM with Rockwell software, demonstrating more advanced agent-assisted utility for controls engineering workflows.
