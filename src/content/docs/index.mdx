Flux is the open-source manufacturing development platform from Green Pipe Partners.

## Install

Run Fluxup with UV on the target Linux host. The installer requires sudo because it creates system users, writes `/opt/flux` and `/etc/flux`, configures Postgres, and installs systemd units.

```bash
sudo "$(command -v uvx)" fluxup init
```

Python fallback:

```bash
python3 -m venv /tmp/fluxup && /tmp/fluxup/bin/python -m pip install --upgrade fluxup && sudo /tmp/fluxup/bin/fluxup init
```

## Projects

- [Flux](https://github.com/GreenPipePartners/Flux) is the controls workbench.
- [Fluxup](https://github.com/GreenPipePartners/Fluxup) is the installer CLI.
- [Fluxy](/docs/flux/0.1.0/fluxy/) connects Ignition to Python 3 through a signed Gateway module and the `fluxy-ign` client.
