fluxy is a free open-source Python library for connecting Ignition to Python 3.

Project repository: [Bobby-Miller/Fluxy](https://github.com/Bobby-Miller/Fluxy)

## What it is for

- Moving data between Ignition projects and Python tooling.
- Building small automation paths without turning Ignition into a general-purpose Python app host.
- Supporting Flux workflows where Python is the right tool.
